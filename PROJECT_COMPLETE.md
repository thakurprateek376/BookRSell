## 💾 Database Design

### User Collection
```javascript
{
  _id: ObjectId,
  name: String,
  email: String (unique),
  password: String (hashed),
  city: String,
  phone: String,
  role: String (buyer|seller),
  timestamps: true
}
```

### Book Collection
```javascript
{
  _id: ObjectId,
  title: String,
  author: String,
  price: Number,
  category: String (enum),
  description: String,
  image: String (file path),
  condition: String (enum),
  city: String,
  seller: ObjectId (references User),
  isActive: Boolean,
  timestamps: true
}
```
